# Multi-task Learning CNN
这个demo是针对于行人精细化属性识别的，需要识别行人性别，发长，衣服类别，裤子类别，鞋子类别等等，所以构造了一个多任务的CNN，需要同时识别很多属性。

同时，demo中包含完整的数据读写的应用，可做参考。
